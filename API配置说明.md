## 快速配置步骤

### 1. 获取API密钥
1. 访问：https://www.deepseek.com/
2. 注册账户并登录
3. 进入控制台获取API密钥
4. 复制格式类似：`sk-xxxxxxxxxxxxxxxxxx`

### 2. 配置密钥
打开项目中的 `.env` 文件，将：
```
DEEPSEEK_API_KEY=sk-your-deepseek-api-key-here
```

替换为：
```
DEEPSEEK_API_KEY=sk-你的真实密钥
```

### 3. 重启服务器
保存文件后，重新运行：
```bash
npm start
```
