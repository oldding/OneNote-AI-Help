# OneNote AI数字人助理 - 快速配置指南

## 🚀 快速开始

### 第1步：获取DeepSeek API密钥

1. **访问DeepSeek官网**
   - 打开 https://www.deepseek.com/
   - 点击"开始使用"或"Get Started"

2. **注册账户**
   - 使用邮箱注册新账户
   - 验证邮箱并完成注册

3. **获取API密钥**
   - 登录后进入控制台
   - 找到"API密钥"或"API Keys"选项
   - 点击"创建新密钥"
   - 复制生成的API密钥（格式类似：sk-xxxxxxxxxxxxxxxx）

### 第2步：配置环境变量

1. **打开.env文件**
   - 在项目根目录找到`.env`文件
   - 如果没有，复制`.env.example`并重命名为`.env`

2. **填入API密钥**
   ```
   DEEPSEEK_API_KEY=sk-your-actual-api-key-here
   PORT=3000
   ```

3. **保存文件**

### 第3步：启动应用

#### 方法1：使用启动脚本（推荐）
- 双击 `start.bat` 文件

#### 方法2：使用命令行
```bash
# 开发模式（自动重启）
npm run dev

# 生产模式
npm start
```

### 第4步：访问应用
- 打开浏览器
- 访问：http://localhost:3000
- 开始与您的OneNote AI助理对话！

## 🛠️ 故障排除

### 问题1：API密钥无效
**症状**：显示"API调用错误"或"401 Unauthorized"
**解决方案**：
1. 检查`.env`文件中的API密钥是否正确
2. 确认API密钥没有过期
3. 检查DeepSeek账户余额

### 问题2：端口被占用
**症状**：显示"EADDRINUSE: address already in use"
**解决方案**：
1. 修改`.env`文件中的`PORT=3001`
2. 或关闭占用3000端口的程序

### 问题3：头像不显示
**症状**：看不到头像图片
**解决方案**：
1. 确认`public/onenote.jpg`文件存在
2. 检查图片文件大小（建议小于1MB）
3. 支持的格式：jpg、png、gif

### 问题4：无法连接网络
**症状**：所有AI回复都显示本地知识库内容
**解决方案**：
1. 检查网络连接
2. 确认防火墙设置
3. 检查代理设置
